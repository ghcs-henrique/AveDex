# AveDex
Catálogo interativo de aves desenvolvido na disciplina de
Boas Práticas de Programação.

## Funcionalidades atuais
- menu em repetição;
- mensagem personalizada;
- apresentação inicial de aves;
- listagem de aves cadastradas;
- busca de aves por nome, família, ordem e dieta;
- exibição de detalhes de uma ave;
- tratamento de opção inválida;
- normalização de texto (ignora acentos e maiúsculas/minúsculas).
